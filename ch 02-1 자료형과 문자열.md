프로그래밍에서 프로그램이 처리할 수 있는 모든 것을 자료(data)라고 부른다. 

### 자료형과 기본 자료형

- 문자열(string) : 메일 제목, 메세지 내용 ("안녕하세요", "Hello World")
- 숫자(number) : 물건의 가격, 학생의 성적 (52, 273, 103.32)
- 불(boolea) : 친구의 로그인 상태 (True, False)

### 자료형 확인하기

자료형(data type)이란 자료의 형식을 말한다. 파이썬에서 자료의 형식을 확인할 때는 type() 함수를 사용한다. print() 함수와 마찬가지로 식별자 뒤에 괄호가 있으므로 함수다. 

```
>>> print(type("안녕하세요"))
<class 'str'>
>>> print(type(273))
<class 'int'>
```

"안녕하세요"라는 문자열을 괄호 안에 넣으니 <class 'str'>이, 숫자를 괄호 안에 넣으니 <class 'int'>가 출력된다. 

### 문자열 만들기

1. 큰 따옴표로 문자열 만들기

문자열은 문자들을 ""로 감싸서 만든다. 
```
>>> print("안녕하세요")
안녕하세요
```

2. 작은따옴표로 문자열 만들기

작은따옴표로'도 문자열을 만들 수 있다.

```
>>> print('안녕하세요')
안녕하세요
```

큰따옴표와 작은따옴표의 결과는 같다. 

### 문자열 내부에 따옴표 넣기

"안녕하세요"라고 말했습니다. 출력하려면

```
>>> print('"안녕하세요"라고 말했습니다.')
"안녕하세요"라고 말했습니다.
```
라고 출력해야한다. ""안녕하세요"라고 말했습니다"를 () 안에 넣으면 구문 오류(Syntax Error)이 나타난다.

### 문자 선택 연산자(인덱싱)[]

문자 선태 연산자는 문자열 내부의 문자 하나를 선택하는 연산자다. 대괄호[] 안에 선택할 문자의 위치를 지정하며, 이 숫자를 인덱스(index)라고 한다. 

파이썬은 숫자를 0부터 세는 제로 인덱스(zero index)다. 

##### 예제 - 문자 선택 연산자의 결과 출력하기
```
print("문자 선택 연산자에 대해 알아볼까요?")
print("안녕하세요"[0])
print("안녕하세요"[1])
print("안녕하세요"[2])
print("안녕하세요"[3])
print("안녕하세요"[4])
```

- 실행 결과
```
문자 선택 연산자에 대해 알아볼까요?
안
녕
하
세
요
```

##### 예제 - 뒤에서부터 선택하기
```
print("문자를 뒤에서부터 선택해 볼까요?")
print("안녕하세요"[-1])
print("안녕하세요"[-2])
print("안녕하세요"[-3])
print("안녕하세요"[-4])
print("안녕하세요"[-5])
```

- 실행 결과
```
문자를 뒤에서부터 선택해 볼까요?
요
세
하
녕
안
```

### 문자열 범위 선택 연산자(슬라이싱) : [:]

문자열의 특정 범위를 선택할 때 사용하는 연산자도 있다. 

```
>>> print("안녕하세요"[1:4])
녕하세
```
파이썬은 마지막 숫자 포함하지 않음을 적용해 출력한다. 위와 같은 코드는 1번째 + 2번째 + 3번째 글자까지만 추출된다. 

문자열 범위 선택 연산자는 대괄호 안에 넣는 숫자 둘 중 하나를 생략하여 다음과 같은 형태로도 사용할 수 있다. 

```
[1:]
[:3]
```
[1:]의 경유 뒤의 값을 생략했기 때문에 1번째부터 끝의 문자를 선택하고 [:3]의 경우 앞의 갚을 생략했기 때문에 0번째부터 뒤의 숫자 3번째 앞의 문자까지 선택한다. 

```
>>> print("안녕하세요"[1:4])

녕하세
>>> print("안녕하세요"[:3])
      
안녕하
```

[]연산자를 이용해 문자열의 특정 위치에 있는 문자를 참조하는 것을 인덱싱(indexing)이라 하고, [:] 연산자를 이용해 문자열의 일부를 추출하는 것을 슬라이싱(slicing)이라 합니다. 

### IndexError(Index out of range) 예외

IndexError 예외는 리스트 / 문자열의 수를 넘는 요소 / 글자를 선택할 때 발생한다. 

```
>>> print("안녕하세요"[10])
```
- 실행 결과
```
Traceback (most recent call last):
  File "<pyshell#9>", line 1, in <module>
    print("안녕하세요"[10])
IndexError: string index out of range 
```
코드를 작성하다가 이런 예외가 발생하면 '리스트/문자열의 수를 넘는 부분을 선택했구나.'라고 알 수 있다. 

### 문자열 길이 구하기

문자열의 길이를 구할 때는 len() 함수를 사용한다. 괄호 내부에 문자열을 넣으면 '문자열에 들어있는 문자의 개수(=문자열의 길이)'를 세어준다. 

```
>>> print(len("안녕하세요")) 
5
```
